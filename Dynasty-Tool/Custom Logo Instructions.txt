1.) Open the templates folder and then open "Logo Template.PSD"
2.) Double-click the smart object named "namehere" and a new window will open.  Drop your custom logo on this layer and CTRL+S to save the smart object
3.) Close the smart object and return to "Logo Template.PSD".  You should now see your custom logo.  Save as "logo.PNG"
4.) Drop "logo.PNG" into the "custom" folder and rename it according to the list below
5.) Open the templates folder and then open "Playoff Logo Template.PSD"
6.) Double-click the smart object named "<--DOUBLE CLICK" and a new window will open
7.) Double-click the smart object named "<--DOUBLE CLICK (LOGO)" and a new window will open.  Double-click that smart object as well.  Drop your custom logo on this layer and CTRL+S to save the smart object
8.) Close the smart object and return to the previous smart object which should now show your logo.  CTRL+S to save this smart object
9.) Close the smart object and return to the previous smart object which should now show your logo with a red background
10.) Double-click the smart object named "<--DOUBLE CLICK (COLOR)" and a new window will open.  Use the paint bucket tool to fill the "CHANGE COLOR HERE" layer with your custom color and CTRL+S to save the smart object
11.) Close the smart object and return to the previous smart object which should now show both your custom logo and your custom color background.  CTRL+S to save the smart object
12.) Close the smart object and return to "Playoff Logo Template.PSD".  You should now see your completed custom Playoff Logo.  Save as "playoff.PNG"
13.) Move "playoff.png"into "playoffs" folder and rename it according to the list below
